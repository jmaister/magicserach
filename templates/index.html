Welcome!

Magic Deep Search is a search engine for Magic: The Gathering cards.

The cards have been tagged using Machine Learning so it is easier to:

  • Find combos
  • Find a card with specific ability
  • etc...

Besides the typical search for the card name, card text, mana color, ... you can search for:

  • Triggers: search by a triggering action i.e. ON_ATTACK -> "Whenever XXX attacks"
  • Effects: search by effect i.e. LIFE_GAIN -> "You gain 3 life.

Note: work is still in progress, so not all card properties or abilities have been trained.
